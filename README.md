# 🥛 Dairy Tracer

🎓 **College Mini Project**

Dairy Tracer is a web-based application developed as part of a **college mini project**.
It allows users to buy and sell fresh dairy products through a simple and interactive interface.

---

## 🚀 Features

### 👤 User Authentication

* Register new account
* Login / Logout system
* Role-based access (User & Admin)

### 🛒 Product Management

* Browse dairy products (Milk, Cheese, Butter, Curd, etc.)
* Search and filter products
* View product details

### 🧺 Cart System

* Add to cart
* Update quantity
* Remove items
* View total bill with tax

### 💳 Checkout & Payment

* Multiple payment methods:

  * Card
  * UPI
  * Net Banking
* Order confirmation & invoice generation

### 📦 Orders

* View order history
* Track order details

### 🛍️ Sell Products

* Users can list their own dairy products
* Manage (edit/delete) their products
* View sales history
---
